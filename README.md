# Nestle Operations Application

A single-page application microsite for Sumant Duggal's Nestle Operations & Supply Chain application.

## Files

- `index.html` - the microsite
- `interactive-cover-letter.html` - the one-page digital cover letter source
- `Nestle Operations Application.pdf` - exportable digital cover letter
- `nestle-product-cluster.png` - hero product cluster asset
- `site-preview.png` - cover letter preview image
- `Sumant Duggal.pdf` - downloadable CV linked from the page

Open `index.html` directly in a browser, or publish the folder with GitHub Pages.
